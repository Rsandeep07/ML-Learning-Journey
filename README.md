# ML-Learning-Journey

Daily hands-on Machine Learning notebooks exploring data preprocessing, EDA, model building, and performance analysis.

------------------------------------------------------------------------------------------------------------------------------------------------------

## About
This repository documents my **daily learning journey in Machine Learning**.  
Each notebook focuses on one concept or dataset, combining both **theory and practical implementation** using Python libraries like `pandas`, `numpy`, `matplotlib`, and `scikit-learn`.

The goal is to build a solid foundation in:
- Exploratory Data Analysis (EDA)
- Data Preprocessing & Feature Engineering
- Supervised and Unsupervised Learning
- Model Evaluation and Tuning
- Real-world ML Project Workflows

-----------------------------------------------------------------------------------------------------------------------------------------------------
##  Tools and Libraries
- **Programming Language:** Python   
- **Data Handling:** Pandas, NumPy  
- **Visualization:** Matplotlib, Seaborn  
- **Machine Learning:** Scikit-learn  
- **Environment:** Jupyter Notebook  

---

## Learning Goal
To develop a strong understanding of **machine learning algorithms, model evaluation, and end-to-end ML workflows** through consistent daily practice.
